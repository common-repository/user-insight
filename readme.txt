ヒートマップ対応アクセス解析ツールUser InsightのWordPressプラグインです。簡単な設定ですぐにUser Insightでデータを計測できるようになります。

== Description ==

# 説明
このプラグインはWordPressのデータをアクセス解析ツールUser Insightで計測するためのプラグインです。
データを解析するため、このプラグインは nakanohito.jp のドメインと通信します。
1分程度の簡単な設定でWordPressページにUser Insightのタグを設置できます。  
<a href="https://ui.userlocal.jp">User Insightについてより詳しく見る</a>

== Installation ==

1. プラグインをインストールします。
2. User Insightにログインして画面右上の解析IDを確認します。
3. 解析IDをプラグイン上で登録します。
4. 解析が始まるのでしばらくすると、訪れたユーザーの解析結果が見られるようになります。
