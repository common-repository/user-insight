== Description ==

User Insight is heatmap analytics tool.
This plugin can place User Insight's tracking code
into your website so that you do not need to add this manually.
It takes only 1 minute to set up the tags of User Insight on the WordPress pages.
This plugin calls external JavaScript deployed on nakanohito.jp to collect web analytics data.
Visit <a href="https://ui.userlocal.jp">User Insight</a> to learn more.

== Installation ==

1. Install plugin.
2. Login to User Insight and check the analytics ID on top right of the screen.
3. Register the analytics ID.
4. Check report after accumulating site access.
